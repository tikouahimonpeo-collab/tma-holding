## 🌍 Google Business Profile — checklist

Google Business Profile (ex "Google My Business") est **critique** pour le
SEO local. Il fait apparaître TMA Holding sur Google Maps et dans le "Local
Pack" (les 3 fiches qui s'affichent en haut des résultats locaux).

### Création (30 min)

1. Va sur [google.com/business](https://www.google.com/business)
2. Clique **Gérer mon entreprise**
3. Remplis :
   - **Nom** : `TMA Holding`
   - **Catégorie principale** : `Entreprise de construction`
   - **Catégories secondaires** : `Magasin de matériaux de construction`, `Rénovation`, `Architecte`
   - **Adresse** : `Cocody Riviera Palmeraie, Abidjan, Côte d'Ivoire`
   - **Zone de service** : ajouter Abidjan + villes secondaires
   - **Téléphone** : `+86 132 65 91 24 68` + `+225 05 85 31 81 52`
   - **Site web** : `https://tma-holding.net`
   - **Horaires** : Lun–Ven 08h–19h · Sam 08h–17h

4. **Validation** : Google envoie un code par courrier postal (1-2 semaines) ou parfois par vidéo/téléphone.

### Après validation

- [ ] Ajouter 10-20 **photos** (chantiers, équipe, bureau, logo)
- [ ] Publier 1 **post** / semaine (nouveau chantier, promo TMA Shop, conseil BTP)
- [ ] Activer la **messagerie** pour les demandes directes
- [ ] Ajouter les **produits** phares de TMA Shop (photos + prix indicatifs)
- [ ] Demander à chaque client satisfait un **avis Google** (5 étoiles)

**Objectif 90 jours** : 20+ avis, fiche 100 % remplie, 50+ photos.
