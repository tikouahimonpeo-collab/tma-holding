# TMA Holding — Prompts Nanobanana prêts à coller

Chaque bloc correspond à **une image** du site. Mode d'emploi :

1. Ouvre Nanobanana (ou Gemini / Google AI Studio).
2. Copie le prompt dans la zone de texte.
3. Génère l'image (format paysage 16:9 pour les héros, 3:2 pour les cartes, 1:1 pour les portraits).
4. Télécharge l'image et renomme-la comme indiqué sur la ligne `Fichier`.
5. Dépose-la à l'emplacement `Chemin` (sous `/public/`).

**Style baseline** (déjà inclus dans chaque prompt) :
photoréaliste · lumière naturelle maîtrisée · rendu 4K net · ambiance premium · couleurs sobres (bleu nuit, gris acier, blanc, touches bleu électrique) · pas de texte/logo/filigrane · contexte Côte d'Ivoire / Afrique de l'Ouest · aucun cartoon ni surdesign.
