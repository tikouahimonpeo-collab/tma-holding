## RÈGLES GLOBALES CANVA (à appliquer systématiquement)

1. **Logo TMA** toujours visible (coin ou centre) · taille mini 80px sur carré, 100px sur story
2. **Zones sûres** :
   - IG feed : marges 60px min
   - Stories : haut 250px (UI) + bas 300px (UI/CTA) réservés
   - TikTok : haut 150px + bas 400px + droite 150px (icônes) réservés
3. **Contraste texte** : AA minimum (ratio 4.5:1) — jamais texte clair sur photo claire sans overlay
4. **Photos** : uniquement chantiers TMA réels · jamais stock générique
5. **Or `#D4A84B`** : usage rare · uniquement accent mot-clé ou élément premium
6. **CTA** : toujours un seul par visuel · verbe d'action + bénéfice
